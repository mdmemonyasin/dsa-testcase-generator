Problem Statement
Given an N x N chessboard and the position of a Knight, find the minimum number of moves required for the Knight to reach the given target position.
The knight moves in a standard chess L-shape:
2 steps in one direction and 1 step perpendicular to it.
The positions are given using 1-based indexing.
If the knight is already at the target position, return 0.

Examples:

Input:
n = 3
knightPos[] = [3, 3]
targetPos[]= [1, 2]

Output: 1

Explanation:
The knight requires just one move to travel from (3, 3) to (1, 2).

Input:
n = 6
knightPos[] = [4, 5]
targetPos[] = [1, 1]
Output: 3

Explanation:

The knight needs 3 moves to reach (1, 1) from (4, 5):
(4, 5) -> (5, 3) -> (3, 2) -> (1, 1).

Constraints:
1 ≤ N ≤ 1000
1 ≤ X, Y ≤ N
1 ≤ TX, TY ≤ N

Custom Input Format:

The first line of input should contain the integer N.
The second line should contain the X and Y coordinates of the knight's starting position.
The third line should contain the TX and TY coordinates of the target position.

Example:
6
4 5
1 1

Output Format:
Print a single integer — the minimum number of moves required.

Example:
3
